# Platform_GAME

<ul>
  <li>Управляйте персонажем с помощью клавиш (стрелочки)</li>
  <li><strong>Соберите ВСЕ золотые монеты на уровне</strong></li>
  <li>После сбора всех монет автоматически начнется следующий уровень</li>
  <li>На каждом уровне больше монет и сложнее препятствия</li>
  <li>Избегайте красных препятствий, чтобы не потерять здоровье</li>
  <li>Синие щиты дают временную защиту</li>
</ul>

<p>Для запуска игры нажмите кнопку "Начать игру"</p>
